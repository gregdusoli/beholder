# beholder

Projeto prático do curso Beholder 3.0 da Luiz Tools cursos, com as seguintes modificações em relação ao projeto original:

- Adoção de Typescript em lugar de JS Vannilla (backend e frontend)
- Definição clara de tipagens (backend e frontend)
- Adoção do `yarn` em lugar do `npm` para gestão de dependências (backend e frontend)
- Dockerização da infraestrutura (backend)
- Integração entre Sequelize e Typescript através da toll `tsx` (backend)
- Adoção de Postgres em lugar de MySQL (backend)
- Adoção de path aliases para melhoria no mecanismo de importações (backend e frontend)
- Separação da estrutura de roteamento (backend)
